Учебен проект за курса "Динамични интернет страници" - Технически университет София

## Описание

Статичен електронен магазин за книги, изграден с HTML, CSS и JavaScript.

## Структура на проекта

```
Final-Project/
├── index.html              # Главна страница с каталог от 5 книги
├── product1.html           # Малкият принц
├── product2.html           # 1984
├── product3.html           # Хари Потър и Философският камък
├── product4.html           # Престъпление и наказание
├── product5.html           # Под игото
├── css/
│   └── styles.css          # Външен CSS файл с всички стилове
├── js/
│   └── script.js           # JavaScript функционалност за формите
└── images/
    └── book1-5.jpg         # Изображения на книгите
```

## Функционалности

### Главна страница (index.html)

- Показва 5 книги като картички
- Всяка картичка съдържа: име, кратко описание, цена и снимка
- Линк към детайлна страница за всяка книга

### Продуктови страници (product1-5.html)

- Пълна информация за книгата (име, описания, цена, снимка)
- Форма за поръчка (име, адрес, бутон)
- Секция с потребителски ревюта
- Форма за добавяне на ново ревю

### JavaScript функционалност

- **Форма за поръчка**: При изпращане показва съобщение "Име, благодарим за поръчката!"
- **Форма за ревю**: Добавя новото ревю най-отгоре в списъка и нулира формата

## Технологии

- HTML5 (семантични елементи)
- CSS3 (Flexbox, основни стилове)
- JavaScript (DOM манипулация, event listeners)

## Стартиране

Отворете `index.html` в уеб браузър.
